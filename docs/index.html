Hey, thanks for checking out my project!

Since SeekhanaMobile is a React Native application, it has to be compiled locally and run on your device.

  1. Download the Expo app on your smartphone if you don't already have it:

    • Google Play Store
    • App Store (iDevices)

  2. Clone my repo onto your system by running the following command in git-bash or your terminal:

    $ git clone https://github.com/wilm42/seekhanaMobile.git

  3. Once the clone is complete, cd into the directory and install the dependencies:

    $ cd seekhanaMobile && npm install
  4. Once the dependencies are installed, run the packager: 
    $ npm start
  5. The packager may take some time to get going. Once it is running, you should see a QR code in your terminal. Open the Expo app on your phone and scan that QR code. This will take some time, as the packager now has to compile my javascript into code native to your device. Once it is complete, Expo should load up the app and you should be able to demo away!
    6. Should you encounter any bugs that I haven't addressed in the known issues, feel free to let me know via email or GitHub Issues.

    Thanks!
